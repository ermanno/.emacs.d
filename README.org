** Org mode LaTeX configuration

Install =latex= and =dvisvgm=

For Ubuntu:

#+begin_src bash
sudo apt install texlive-latex-base dvisvgm
#+end_src

For Fedora:

#+begin_src bash
sudo dnf install texlive-scheme-basic dvisvgm
#+end_src

** Rust
Install rust-analyzer with:

#+begin_src bash
rustup component add rust-analyzer
#+end_src

Install rust-script (for Org Babel) with:

#+begin_src bash
cargo install rust-script
#+end_src

Tree-sitter on Mac:

#+begin_src bash
# Create the directory
mkdir -p ~/.emacs.d/tree-sitter

# Clone the grammar, build the shared library, and install it
git clone --depth 1 https://github.com/tree-sitter/tree-sitter-rust /tmp/tree-sitter-rust
cc -shared -o ~/.emacs.d/tree-sitter/libtree-sitter-rust.dylib \
   -I /tmp/tree-sitter-rust/src \
   /tmp/tree-sitter-rust/src/parser.c /tmp/tree-sitter-rust/src/scanner.c
rm -rf /tmp/tree-sitter-rust
#+end_src
